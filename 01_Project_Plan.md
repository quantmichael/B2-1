# 서비스 기획

---

## 1. 프로젝트 개요

### 프로젝트명

**COSMOS DAILY**

### 슬로건

**Explore the Universe Every Day**

### 프로젝트 한 줄 소개

> NASA Open API를 활용하여 매일 새로운 우주 사진과 천문 정보를 쉽고 아름답게 제공하는 모바일 애플리케이션
> 

---

## 2. 프로젝트 배경

NASA는 누구나 사용할 수 있는 Open API를 제공합니다.

하지만 일반 사용자가 NASA 홈페이지를 방문하여 원하는 정보를 찾는 것은 다소 어렵고, 모바일 환경에서도 사용성이 높다고 보기 어렵습니다.

COSMOS DAILY는 NASA의 데이터를 누구나 쉽고 직관적으로 감상할 수 있도록 모바일 UX 중심으로 재구성한 서비스입니다.

---

## 3. 프로젝트 목표

이번 프로젝트의 목표는 다음과 같습니다.

- NASA Open API를 활용한 실제 서비스 기획
- AI를 이용한 UI 디자인 제작
- Figma를 이용한 프로토타입 제작
- 사용자 친화적인 모바일 UX 설계

# 5. 핵심 기능

이번 앱은 기능을 많이 넣지 않습니다.

과제는 **UI/UX 설계**가 핵심이므로 꼭 필요한 기능만 넣겠습니다.

| 기능 | 설명 |
| --- | --- |
| 오늘의 우주 | NASA APOD API를 이용한 오늘의 사진 |
| 날짜 검색 | 원하는 날짜의 우주사진 조회 |
| 상세 보기 | 사진과 설명 확인 |
| 즐겨찾기 | 마음에 드는 사진 저장 |
| 설정 | 다크모드 및 앱 정보 |
